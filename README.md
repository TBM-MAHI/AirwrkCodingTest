Airwrk Takehome Test
==============================
## Installation

##### Run the following commands to run this project on localhost

```sh
git clone https://https://github.com/TBM-MAHI/AirwrkCodingTest
```
```sh
cd AirwrkCodingTest
```
```sh
npm install
```
##### run in watch mode
```sh
npm run watch
```
##### Run the API test 
```sh
npm run test
```
